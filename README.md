# ITB Resource Calculator

A web-based tool designed to simulate, analyze, and optimize resource consumption in an educational environment.

## Overview

The ITB Resource Calculator allows users to estimate and visualize the consumption of:

* Water
* Electricity
* Office supplies
* Cleaning supplies

It also includes simulation features to test cost-saving strategies and sustainability measures.

## Features

* Interactive calculator with configurable parameters
* Seasonal consumption adjustments
* Holiday calendar integration
* Data visualization and historical analysis
* Sorting and filtering of datasets
* Simulation of saving measures with charts (Chart.js)
